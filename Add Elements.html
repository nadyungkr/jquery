append() -선택한 요소의 끝에 내용을 삽입
        prepend() -선택한 요소의 시작 부분에 내용을 삽입
        after() -선택한 요소 뒤에 내용을 삽입
        before() -선택한 요소 앞에 내용을 삽입

        위append()및 prepend()메소드 모두 무한한 수의 새로운 요소를 매개 변수로 사용할 수 있음
        새로운 요소는 텍스트 / HTML,
        jQuery 또는 JavaScript 코드 및 DOM 요소로 생성 할 수 있음
        
        after() 및 before()도 마찬가지
        after, before의 경우 이미지 같은 요소 뒤에 사용함

여기에 문장을 덧붙여 볼까

  1. list item 1
  2. list item 2
  3. list item 3

append, prepend를 html,jquery,dom에 새로 추가하여 써보기
